# ShibutzPlus

A modern Next.js 14+ application built with TypeScript and CSS.

## Project Overview

ShibutzPlus is built with the following technologies and features:

- Next.js 14+ with App Router
- TypeScript for type safety
- CSS (no Tailwind)
- ESLint for code quality
- Prettier for code formatting
- Organized project structure with src directory

## Project Structure

```
src/
├── app/         # App Router pages and layouts
├── components/  # Reusable UI components
```

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

- Edit pages in `src/app/`
- Create reusable components in `src/components/`
- Modify styles in CSS files

## Code Quality

This project uses ESLint and Prettier for code quality and formatting:

```bash
# Run ESLint
npm run lint

# Format code with Prettier
npm run format
```
